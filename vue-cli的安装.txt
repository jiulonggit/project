1、安装node.js：
     下载-->安装
     验证node.js：
      	进入cmd命令行
     	node -v
     	npm -v
2、安装淘宝镜像
     npm install -g cnpm --registry=https://registry.npm.taobao.org
     验证：
	cnpm -v
3、安装vue-cli脚手架
     cnpm install -g @vue/cli
     验证：
	vue -V
4、创建vue项目
     进入项目存放的文件夹
 	切换盘符：e:
	切换目录：cd \aaa\bbb
     创建vue项目
    	vue create <项目名称>	--字符界面
	vue ui			--图形化界面

5、运行vue项目
     进入项目所在的文件夹：cd  <项目文件夹>
     运行项目：npm run serve
     查看项目：浏览器：http://localhost:8080
